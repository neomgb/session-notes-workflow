设置用户名和邮箱：
```shell
git config --global user.name "Your Name"
git config --global user.email "email@example.com"
```

使用了 `--global` 选项，所有的 Git repo 都会使用这个配置

PS：如果想针对特定项目使用不同的用户名与邮箱时，运行没有 `--global` 选项的命令来配置
